## 先搞清楚：DMIT 便宜在哪，不便宜在哪

DMIT 是做精品网线路出身的。他家做了七八年，主打的卖点就两件事：不超售、线路质量稳。全系标配 AMD EPYC 处理器，这个配置在同价位里不多见。

**便宜的部分**：T1 系列（国际线路）年付套餐，洛杉矶、香港、东京三个机房都有，起步价 $36.9/年，算下来每月三块多美元。流量用完不切断，降速继续跑，这点对流量大的场景很友好。

**不便宜的部分**：Pro 系列（CN2 GIA 三网优化）和 Eyeball 系列（CMIN2/9929 优化）定价明显高一截。香港 Pro 最低套餐月付要 $39.9，日本和洛杉矶略便宜，但也不是三五块钱的事。

所以"DMIT 便宜"这件事的正确理解是：**T1 系列有性价比入口，优化线路走量的话要配合优惠码才划算**。

## 什么是 T1、Pro、Eyeball——一分钟看懂三条线路

很多人被 DMIT 的产品命名搞晕，这里直接说人话。

**Pro 系列**：三网全程 CN2 GIA 优化。电信、联通、移动回程都走 GIA。延迟低，晚高峰稳定，是面向国内用户体验最好的线路。价格最贵。

**Eyeball 系列**：联通和电信走 9929/CUG 高级线路，移动走 CMIN2 双程。整体比 Pro 便宜一些，但对联通移动用户来说体验已经很不错了。性价比选项。

**T1 系列**：国际线路，没有针对国内三网的专门优化。面向海外用户建站、亚洲内部低延迟互联、或者不需要中国优化的业务场景。价格最低，流量最大，超量后降速不断服务。

## 选哪个机房比较好？

这个没有标准答案，主要看你的用户在哪、你的业务方向。

**主要受众是中国大陆用户**：洛杉矶或香港 Pro / EB 系列。香港延迟更低（实测通常 20-50ms），洛杉矶价格更友好，两者都有 CN2 GIA 可选。

**面向日本或东亚地区**：东京机房，T1 系列接入了 GSL 线路，亚太互联质量不错，价格也有竞争力。

**主要是国际业务**：三个机房的 T1 系列都可以。洛杉矶 T1 VOLUME 系列流量给得大，10Gbps 带宽，适合流量密集型用途。

## 几个容易踩的坑

1. **不要月付买便宜套餐**。大部分优惠码只对季付或年付生效，月付通常没折扣。如果你打算长期用，算一下年付比月付能省多少，一般很明显。

2. **套餐经常缺货**。DMIT 不超售这件事是认真的，热门套餐真的会卖完。看到心仪配置有库存，不要太犹豫。

3. **T1 系列没有国内优化**。不少人看到价格便宜就下单，结果发现晚高峰国内访问很慢——这不是故障，是产品设计如此。如果你的主要用户在国内，要选 Pro 或 EB 系列。

4. **AN4 和 AN5 平台不一样**。AN5 用的是更新一代的 AMD EPYC 9005 处理器，性能更高，但价格也高一截。个人轻量用途 AN4 完全够，不用非选最新。

**Q：流量用完会被断网吗？**
不会。T1 系列超量后限速到一定带宽（100Mbps 左右，具体看套餐），但服务不中断。Pro 和 EB 系列有单独的策略，以官网套餐页说明为准。

**Q：IP 被墙怎么办？**
Premium 和 Eyeball 系列支持免费换 IP，条件是 IP 确认被墙，每 15 天可以申请一次，超出频次收费 $5 一次。T1 系列不保证 IP 可用性，换 IP 要另外收费。

**Q：优惠码可以叠加用吗？**
不能。每次下单只能用一个优惠码。选优惠力度最大的那个用就行。
